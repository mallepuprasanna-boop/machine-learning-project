# ⚙️ Installation

Clone the repository:

git clone https://github.com/Amitsinghyt/Darshan_Ease.git

Go to the project folder:

cd Darshan_Ease

Install dependencies:

npm install

---

# ▶️ Run the Project

Start backend server:

npm start

Start frontend:

npm run dev

Open browser:

http://localhost:3000
